Instruction

  • Add new entries below or edit the configuration directly to set the paths you want to use for Pathmarks.
  • Click the browser action icon
  • Choose the path you would like to open. The url of your current tab will be used to create the new path url.
  • You can add, remove and reorder the entires in the popup.
  • Pathmarks switches to the newly created tab. If you use the Shift-Key when choosing a Pathmark the tab will be opened but the focus stays on the current tab.
  • You can configure a hotkey in the Extensions settings to use Pathmarks with your keyboard.
    • Navigate with up/down keys
    • Open an entry using enter and use the Shift key if you want keep the focus.

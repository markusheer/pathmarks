PathMarks configuration

  • Create a JSON configuration to set the paths you want to use for PathMarks.
  • Click the browser action icon
  • Choose the path you would like to open. The url of your current tab wil be used to create the new path url.

JSON Example

[{"title": "CMS Admin", "value": "/admin"},
{"title": "CMS Author", "value": "/author"}]

Use textarea to edit the configuration.
